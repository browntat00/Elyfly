## Deploy to Vercel

1. Push this folder to a new GitHub repo.
2. Import the repo at vercel.com/new and deploy.
3. In Vercel project settings, add your custom domain `elyfly.com` (or `shop.elyfly.com`) and follow the DNS instructions.
4. Done!
